# Infrastructure Module

This folder stores infrastructure assets separated by concern:

1. `docker/`: local image definitions and container build instructions.
2. `azure/`: Azure deployment templates and environment mapping notes.
3. `github-actions/`: CI/CD workflow templates before promoting to `.github/workflows`.

## Files

1. `docker/README.md`: explains Dockerfiles used during local development.
2. `azure/README.md`: explains planned Azure deployment artifacts.
3. `github-actions/README.md`: explains workflow template organization.
