# Layer 3: Agents Module

This folder will contain the GenAI stack, including retrieval, prompt orchestration, and agent tools.

## Files

1. `__init__.py`: package marker for agentic modules.

## Planned Additions

1. `retrieval/`: vector search and document ranking with Qdrant.
2. `prompts/`: prompt templates and system instruction management.
3. `orchestrators/`: multi-step tool-using agent flows.
4. `providers/`: local Ollama adapters and Azure OpenAI adapters.
